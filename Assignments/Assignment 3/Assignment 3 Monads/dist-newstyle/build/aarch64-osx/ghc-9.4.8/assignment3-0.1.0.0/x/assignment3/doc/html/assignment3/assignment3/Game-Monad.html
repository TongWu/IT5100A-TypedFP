type RandomSeed = Integer #

The seed for random number generation. We use Integers for seeds

type GameState = (GameBoard, RandomSeed) #

The GameM monad uses a GameBoard and a RandomSeed as its state.

type GameM = StateT GameState IO #

The GameM monad is a StateT monad transformer over the IO monad that uses GameState for its state. Thus, we can use the StateT methods like put, get and evalStateT on it.
