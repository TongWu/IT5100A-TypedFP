Description

This module contains all data types supporting a game of tic-tac-toe. In particular, it is a game where humans are always first to play.

The GameBoard

data GameBoard #

Game boards

A tic-tac-toe game board. The constructor should be treated as private unless within the Game module. Most operations should make use of blankBoard, a GameBoard with no pieces. There is also testBoard which provides a GameBoard for testing only.

setBlank #

Arguments

:: Int

n, i.e. the position to set

-> GameBoard

The GameBoard to set

-> GameBoard

The resulting GameBoard where the nth position is set with the current player's piece

Sets the nth blank position in the GameBoard with the current player's piece. If n is out of bounds, then the GameBoard is returned unmodified.
